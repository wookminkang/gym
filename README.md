# 건강 및 피트니스를 활용한 어플

### 주요 기능

> 앱에서 제공하는 체성분 데이터는 17가지로 아래와 같습니다.
>> 1. 체중
>> 2. BMI
>> 3. 체지방율
>> 4. 내장지방지수
>> 5. 수분
>> 6. 골격근량
>> 7. 근육량
>> 8. 골질량
>> 9. 단백질
>> 10. 기초대사량 (BMR)
>> 11. 신체나이
>> 12. 지방량
>> 13. 체수분(kg)
>> 14. 체단백질
>> 15. 비만 정도
>> 16. 표준 체중
>> 17. 체형
